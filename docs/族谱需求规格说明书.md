# 族谱（Zupu）产品需求规格说明书

> 版本：v1.0  
> 对标网站：MyHeritage、Ancestry、FamilySearch、23andMe、族谱网  
> 设计风格：Apple 毛玻璃磨砂 + 中国古典元素融合

---

## 一、产品定位

**一句话描述：** 面向中国家庭的现代化家族谱系管理平台，融合传统谱式格式与现代交互体验。

**目标用户：**
- **族长/族老**（50-70岁）：管理整族谱牒，审核修改，最终打印纸质谱书
- **中年族人**（35-50岁）：维护自家支系，上传照片和大事记，联络宗亲
- **年轻族人**（18-35岁）：浏览家族树，查看祖源故事，参与家族互动

## 三、UI/UX 设计要求

### 3.1 整体风格

```
风格定位：现代东方美学 = Apple 玻璃质感 + 中国传统元素点缀

基础：毛玻璃磨砂风格（当前已实现）
主色调：深色底（#000-#1a1a2e）+ 金色点缀（#c9a84c）
        浅色底（#f5f5f7）+ 暗红点缀（#8b2500）
字体：无衬线系统字体为主，标题可选书法字体
图标：简洁线性图标（Lucide），辅以传统纹样装饰
```

### 3.2 颜色体系（扩展版）

| 用途 | 色值（Light） | 色值（Dark） | 说明 |
|------|-------------|-------------|------|
| 背景 | `#f5f5f7` | `#0a0a0f` | Apple 灰/深空黑 |
| 卡片 | `#ffffff/55` | `#1a1a2e/50` | 半透毛玻璃 |
| 主文字 | `#1d1d1f` | `#f5f5f5` | - |

| 女性 | `#ec4899` | `#f472b6` | 粉色系 |
| 在世 | 不透明 | 不透明 | - |
| 逝者 | 透明度0.5 | 透明度0.4 | 灰度标识 |
| 世系线 | `#c9a84c` | `#c9a84c` | 金色连线 |

| 警告 | `#ff9f0a` | `#ff9f0a` | Apple 橙 |

### 3.3 可视化组件标准

#### 家族树（核心组件）

```
1. 扇形式家谱图 (Fan Chart)
   - 以当前用户为中心，向外辐射
   - 每环代表一代（最多7代）
   - 男性扇区蓝色渐变，女性粉色渐变
   - 逝者扇区透明度降低
   - 点击扇区展开详情
   - 支持触摸旋转

2. 吊线式世系图 (Pedigree Chart)
   - 传统自上而下竖排
   - 夫妻并排，子女在下方居中
   - 金色实线连接配偶关系
   - 金色虚线连接亲子关系
   - 支持无限展开/收拢

3. 时间线视图 (Timeline)
   - 水平时间轴+垂直事件列
   - 按年代排列成员生平节点
   - 可缩放至世纪/年代/年份三级
```

### 3.4 交互规范

| 交互 | 规范 |
|------|------|
| 点击 | 树节点点击→弹出详情浮层（Modal式） |
| 双击 | 双击节点→进入成员编辑模式 |
| 拖拽 | 拖拽人物卡片到目标节点→建立关系 |
| 悬停 | 悬停显示快速预览（姓名+生卒年+头像） |
| 缩放 | 双指捏合/滚轮缩放，平滑过渡 |
| 长按 | 移动端长按→出现快捷操作菜单 |
| 右击 | 桌面端右键→上下文菜单（编辑/删除/添加配偶/添加子女） |
| 搜索 | 实时搜索，输入即搜，结果高亮定位 |

### 3.5 自适应断点

| 断点 | 布局调整 |
|------|---------|
| < 640px | 单列，底部Tab导航，简化树图（仅显示3代） |
| 640-1024px | 双列，侧边栏折叠，完整树图 |
| > 1024px | 三列/全宽，完整功能 |

#### 颜色系统建议（复用 pure-genealogy 设计）

```typescript
// 松柏绿统一色系 — 避免性别红蓝冲突
const UNIFIED_BASE_COLOR = { h: 145, s: 45, l: 30 };
// 每代 +6% 亮度，产生代际层次感
function generateBranchColor(base, generationOffset) {
  const step = 6, maxLightness = 85;
  const lightness = Math.min(base.l + (generationOffset * step), maxLightness);
  return `hsl(${base.h}, ${base.s}%, ${lightness}%)`;
}
```

## 六、竞品差异化策略

| 维度 | 国际竞品弱点 | 我们可做的优势 |
|------|------------|--------------|
| 中文体验 | Ancestry 无中文，MyHeritage 中文生硬 | 原生中文，书法字体，传统谱式 |
| 传统格式 | 只支持国际通用树图 | 苏式/欧式/宝塔式/五服图 |
| 社交属性 | 弱（仅邮件邀请） | 微信分享、宗亲圈、动态流 |
| 打印服务 | 无 | **纸质谱书排版+印刷对接**（核心变现） |
| 价格 | 订阅 $20-40/月 | 基础免费 + 增值服务（打印/AI修复） |
