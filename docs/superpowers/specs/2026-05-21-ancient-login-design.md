# 古风登录页设计文档

## 概述

为族谱应用重新设计登录、注册、忘记密码三个认证页面，采用水墨画卷风格（古风视觉）。

## 设计方向

**A · 水墨画卷** — 仿《富春山居图》风格的长卷山水背景，登录表单以画卷题跋形式融入画面，融合红色印章与古籍边框元素。

### 核心理念

- 背景复用 `InkWashBackground` 组件（Canvas 水墨渲染）
- 登录表单设计为画卷上的书法题跋/落款
- 红色印章作为按钮和视觉点缀

## 色彩方案

| 色名 | 色值 | 用途 |
|------|------|------|
| 宣纸白 | `#f5e6c8` | 页面主背景色、卡片底色 |
| 绢本色 | `#e0d0b8` | 渐变过渡色 |
| 藤黄褐 | `#c4b08a` | 边框、装饰线 |
| 赭石 | `#8b7d6b` | 标签、副标题 |
| 焦茶 | `#5a4a3a` | 正文、图标 |
| 墨黑 | `#2c2c2c` | 品牌标题 |
| 朱红 | `#c43a31` | 按钮（印章）、强调色 |
| 青灰 | `#7a8a8a` | 辅助信息 |

## 排版

- **标题/品牌** — `Noto Serif SC` 衬线字体，大字距（letter-spacing: 8px）
- **正文/标签** — `Noto Sans SC` 或系统无衬线
- **竖排诗文** — 左侧装饰性竖排书法文字（`writing-mode: vertical-rl`）
- **按钮** — 朱红底色，衬线字体，宽字距（letter-spacing: 6px）

### 桌面端 (≥ 1024px)
- 全幅水墨山水背景（Canvas 渲染）
- 表单卡片位于右侧三分之一处（像画卷题跋）
- 左侧竖排书法诗文装饰
- 右上角有朦胧月亮

### 平板端 (768~1023px)
- 全幅背景，表单居中
- 诗文隐藏

### 手机端 (≤ 767px)
- 简化水墨背景（仅保留一重山）
- 表单全宽
- 所有装饰元素精简

## 组件设计

### 表单卡片
- 仿宣纸底色（`rgba(245,235,215,0.92)`）
- 顶部和底部有渐变装饰线（毛笔笔触效果）
- 轻微圆角 + 柔和阴影
- 半透明边框

### 输入框
- 无边框，仅有底部渐变线（毛笔笔触：两端淡中间深）
- Focus 状态：笔触加深、加粗
- 标签文字在输入框上方（小字，赭石色）

### 登录按钮 (印章风格)
- 朱红底色（`#c43a31`）
- 衬线字体白字，宽字距
- hover：模拟朱砂印泥微凸效果
- active：微压效果

### 底部链接
- 小字（11px）
- 青灰色（`#a09070`）
- hover：如墨迹洇开效果

## 三个页面差异

| 页面 | 表单标题 | 按钮文字 | 底部链接 |
|------|---------|---------|---------|
| 登录 | 登 录 | 登 录 | 注册账户 · 忘记密码 |
| 注册 | 注 册 | 注 册 | 已有账户？登录 |
| 忘记密码 | 重置密码 | 发送验证码 | 返回登录 |

## 实现方案

1. **创建 `AncientAuthLayout`** — 新的认证布局组件，用于替换 `StripeAuthLayout`。包含：
   - 复用 `InkWashBackground` 作为背景
   - 半透明遮罩层
   - 左侧竖排诗文装饰（桌面端）
   - 响应式容器

2. **创建 `AuthFormCard`** — 表单卡片容器组件，统一三个页面的卡片样式：
   - 宣纸底色
   - 顶部/底部装饰线
   - 标题区
   - 插槽式 children

3. **创建 `BrushInput`** — 毛笔笔触风格输入框组件：
   - 支持 label、placeholder、error 状态
   - 底部渐变笔触线
   - Focus 动画
   - 密码可见性切换

4. **创建 `SealButton`** — 印章风格按钮：
   - 朱红底色 + 白字
   - 衬线字体宽字距
   - hover/active 动画

5. **修改三个页面**：
   - `LoginPage.tsx` — 改用新布局，保留表单验证和数据流逻辑
   - `RegisterPage.tsx` — 同上
   - `ForgotPasswordPage.tsx` — 同上

## 数据流

认证逻辑保持不变：
- Redux store（`authSlice`）管理认证状态
- 表单验证逻辑保留（必填项、格式校验）
- `usePasswordToggle` 和 `useRememberMe` hooks 保留
